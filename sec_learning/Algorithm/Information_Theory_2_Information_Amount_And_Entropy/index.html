熵 (平均自信息量)

    信源的 平均自信息量，也称为信源的 熵，信源熵在平均意义上来表征信源的总体特征。

    为了分析信源信息的量化特征，我们需要一个函数用来输出信源的熵，我们称之为 信源熵函数。我们把信源熵函数记为 `H(p_1, p_2, ..., p_a)`，其中 `p_i` 是各个信源符号出现的概率。信息熵函数有时也称为 信息熵函数。

信息熵函数的条件

    在给出信源熵的具体定义之前，我们先来看香农给出的信息熵函数应该满足的三个条件:

  • 连续性: `H(p_1, p_2, ..., p_a)` 是 `(p_1, p_2, ..., p_a)` 的连续函数;
  • 等概时单调递增: `H(\frac{1}{a}, \frac{1}{a}, ..., \frac{1}{a}) < H(\frac{1}{a+1}, \frac{1}{a+1}, ..., \frac{1}{a+1})`
  • 可加性 (信息可积累): 对于任意正整数 `b_i` (`i=1,2,...,k`)
    如果 `\sum_{i=1}^{k}b_i = a`，则有: `H(\frac{b_1}{a}, \frac{b_2}{a}, ..., \frac{b_k}{a})+\sum_{i=1}^{k}\frac{b_i}{a} \cdot H(\frac{1}{b_1}, \frac{1}{b_2}, ..., \frac{1}{b_k})`

    对于第二点 —— 等概递增性是比较显然的: 在等概的条件下，信源符号的个数越多，信源的熵越大。

    最后一点 —— 可加性，可以这么理解: 要寻找我校二年级的一位学生，设 `a` 是该系二年级学生总数，`k` 是该系二年级的班级数，而 `b_i` 是该系全体二年级第 `i` 班的学生人数。则:

二年级全体学生的不确定定性 = 班级的不确定性 + 每个班学生的平均不确定性。

    另外，Khinchin 条件增加了一个极值条件 (对于离散信源):

  • `maxH(p_1, p_2, ..., p_a) = (\frac{1}{N}, \frac{1}{N}, ..., \frac{1}{N}) = \log_{2}a`

信息熵

    现在让我们给出信息熵的定义。在数学上，信息熵 当且仅当 定义为信源中各个符号自信息量的数学期望时，满足上述的性质:

`H(X) = E(I(X)) = \sum p(x_i)\cdotI(x_i) = \sump(x_i)\cdot\log\frac{1}{p(x_i)} = - \sump(x_i)\cdot\logp(x_i)`

    其中 `x_i` 和 `p(x_i)` 取自我们上一篇文章 信息论导论 中提过的信源符号的概率空间，表示信源输出的各个信源符号及其输出概率; `I(x_i) = \log\frac{1}{p(x_i)}` 称为信源符号的 不确定性，可以发现信源符号产生的概率越大，其不确定性越小。

    我们规定当符号概率 `p(x_i)` 为 0 时，`p(x_i)\logp(x_i)` 也为 0。可以简单的理解为: 如果一个信源符号不可能产生，那么它带来的不确定性为 0。

    信息熵表示了信源输出前，信源的平均不确定度，同时也表示了信源输出后，每个符号所提供的平均信息量。

    信息熵函数是一个关于 `p` 的上凸函数，存在一个最大值。

条件熵

    在给定事件 `y_j` 的条件下，`x_i` 的条件自信息量为 `I(x_i|y_i) = -\logp(x_i|y_i)`，其含义是 `y_i` 发生的条件下，`x_i` 所给出的信息量。基于这种定义，我们定义 `X` 集合的 条件熵 为: `H(X|y_j) = f(y_j) = \sum_{i}p(x_i|y_j)I(x_i|y_j)`，`H(X|y_j)` 是 `y_j` 的一个函数。

    相应地，在给定随机变量 `Y` 的条件下，`X` 集合的条件熵为: `H(X|Y) = \sum_{i,j}p(x_i,y_j)I(x_i|y_j)`。值得注意的是，条件熵是在联合符号集合 `XY` 上的条件自信息量的联合概率加权统计平均值，也即 `H(X|Y) = E_{X,Y}()`

联合熵

    联合熵 (共熵) 是联合符号集合 `XY` 上的每个元素对 `x_i,y_j` 的联合自信息量的联合概率加权统计平均值。定义为: `H(XY)=\sum_{i,j}p(x_i,y_j)I(x_i,y_j)=-\sum_{i,j}p(x_i,y_j)\logp(x_i,y_j)`

熵，条件熵，联合熵的关系

    熵，条件熵，联合熵满足以下关系:

  • `H(XY) = H(X) + H(Y|X) = H(Y) + H(X|Y)`
  • `H(XYZ) = H(XY) + H(Z|XY) = H(X) + H(Y|X) + H(Z|XY)`

    熵 `H(X)` 表示 `X` 单独的不确定性；条件熵 `H(X|Y)` 表示观察到 Y 的条件下 X 的不确定性，`H(X|y_j)` 是关于 `y_j` 的一个函数；联合熵 `H(XY)` 表示 X 和 Y 共同的不确定性。

    熵的一些性质总结如下:

  • 条件熵不大于信源熵: `H(X|Y) \leq H(X)`，当 `X` 和 `Y` 相互独立时，式子取等号;
  • 条件上的条件越多，条件熵的值约小: `H(Z|XY) \leq H(Z|X)`，当且仅当 `p(z|xy) = p(z|x)` 时，式子取等号;
  • 联合熵和两个集合的信源熵存在关系: `H(XY) \leq H(X) + H(Y)`，当 `X` 和 `Y` 相互独立时，式子取等号
  • 当信源含有的新增消息为小概率时，熵不变: `\lim_{\epsilon \rightarrow \infty}H(a_1, a_2, ..., a_n-\epsilon, \epsilon) = H(a_1, a_2, ..., a_n)`

相对熵和交叉熵

    两个概率分布为 `P(X)` 和 `Q(X)` 的随机变量的 相对熵 为: `D`(`P`||`Q`) = `\sum_{x\inX}p(x)\log\frac{p(x)}{Q(x)} = H(p,q) - H(p)`，其中 `H(p,q)` 为 交叉熵，定义为 `H(p,q)=\sum_xp(x)\logfrac{1}{q(x)}`。相对熵度量为当真实分布为 `P` 而假定分布为 `Q`时的无效性 (e.g. 已知随机变量的真实分布为 `p`，可构造平均描述长度为 `H(p)` 的编码，如果使用针对分布 `q` 的编码，则在平均意义上就需要 `H(p)+D(p||q)` 比特来描述这个随机变量)。

    相对熵描述的是两种概率分布之间的 "距离" (i.e. 非对称距离，指的是两种分布的差异性)。

    交叉熵可以认为是对预测分布 `q(x)` 用真实分布 `p(x)` 来进行编码所需要的信息量大小。

    相对熵是非对称的: `D(P||Q) \ne D(Q||P)`

    条件相对熵: `D(P(y|x)||Q(y|x)) = \sum_{x\inX}p(x)\sum_{y\inY}p(y|x)\cdot\log\frac{p(y|x)}{Q(y|x)}`

平均互信息量

定义

    定义 互信息量 为 `I(x_i;y_j) = I(x_i) - I(x_i|y_j)`，其物理含义是已知输出为 `y_j` 时，输入 `x_i` 的不确定程度; 同理，`I(y_j;x_i) = I(y_j) - I(y_j|x_i)` 表示的是已知输入为 `x_i` 时，输出 `y_j` 的不确定程度。

    `I(X;Y) = H(X) - H(X|Y) \leq H(X)`，我们可以称 `H(X|Y)` 为 损失熵。

    `I(Y;X) = H(Y) - H(Y|X) \leq H(Y)`，`H(Y|X)` 为 噪声熵

    可以理解为：在看到输出 `Y` 后，信息的不确定性是 `H(X|Y)`，而信源的不确定性是 `H(X)`，因此通信系统的传输帮助减少的不确定性是 `H(X)-H(X|Y)`，这个数值就描述了通信系统的性能。反过来也可以理解，`H(Y)-H(Y|X)` 描述的是在看到知道信源输出以后，系统看到信宿的输出对不确定性的减少量。

    可见，有扰信道的加入只会引入对原始不确定性的衰弱，不可能引入新的信息。

    定义 平均互信息量 为 `I(X;Y)=\sum_{i}\sum_{j}p(x_i,y_j)\cdotI(x_i;y_j)=\sum_{i}\sum_{j}p(x_i,y_j)\cdot\frac{p(x_i/y_j)}{p(x_i)}`。

    信息量只描述信源/信宿的不确定程度，平均互信息量描述的是整个系统的通信不确定程度。

性质

    非负性: `I(X;Y) \ge 0`。

    对称性: `I(X;Y) = I(Y;X)`，它们描述的都是整个通信系统的特性。

    极值性: `I(X;Y) \le H(X)`, `I(X;Y) \le H(Y)`。

多变量条件互信息量

    `I(X;Y|Z)=H(X|Z)-H(X|Y;Z)=H(Y|Z)-H(Y|X;Z)`

信息不等式

    为了方便后续研究传输限和压缩限的问题，我们需要要如下的不等式工具:

  • 基础不等式: `\lnx \le x-1` `(x > 0)`
  • Jensen 不等式: 如果 `f(x)` 为一个下凸函数 (i.e. `f(\lamdax_1+(1-\lamda)x_2) \leq \lamdaf(x_1)+(1-\lamda)f(x_2)`，当 `\lamda=0,1` 时等号成立)，`X` 是一个随机变量，则有 `E(f(x)) \ge f(E(X))`，其中 `E(X)=\sum_{x\inX}p(x)\cdotx` 。
  • 信息散度不等式: `D(P||Q) \ge 0`
  • Fano 不等式: 通信的本质是估计： `\hat{X}=g(Y)`。`\hat{X}` 的错误概率和条件熵 `H(X|Y)` 有关系。我们定义随机变量 `P_e=P{\hat{X} \ne X}` 代表错误概率，我们有: `H(P_e)+P_e\log(|X|-1) \ge H(X|Y)`，其中 `H(P_e)` 为判决结果是否正确的不确定度，`P_e\log(|X|-1)` 为已知发送出错之后，在剩余的 `|X|-1` 种可能结果种的不确定性的最大值 (i.e. 也即符号等概率分布的情况)。
