1. 什么是 Fetch？

    fetch 是 XMLHttpRequest 的一种替代方案，用于 JavaScript 脚本与服务器进行交互。如果你还没有阅读我的另外两篇文章 Async JavaScript 和 AJAX，我推荐您先对 JS 中的异步操作 和 XMLHttpRequest 对象 做下了解。

2. 使用 Fetch

    fetch 是一个 异步操作，可以从指定的 url 获取数据。它的返回值是一个 Promise 对象，这意味着可以使用 Promise 的 then 和 catch 方法等对 fetch 操作获取的内容做后续处理。

    在这个 Promise 对象中封装的内容是一个 Response 对象，里面封装了来自服务器相应的各种信息。
