Modified DPDK API in mTCP

    Some of the DPDK APIs have been updated, hence we need to modify these DPDK API references in mTCP source code. These modifications including:

rte_get_master_lcore $\rightarrow$ rte_get_main_lcore
eal_proc_type_detect $\rightarrow$ rte_eal_process_type
Privatized lcore_config

    The struct lcore_config has been change to be a EAL private struct dpdk_lcore_config, so third-party applications are no longer be able to use it, so we need to comment out some references to this struct: TO CHECK: Does simply comment these code making sense?

Update Deprecated Checksum Flag

    Some flag related to checksum offloading are deprecated, we should modify change them to new flags as listed as follows:

  • PKT_RX_L4_CKSUM_BAD $\rightarrow$ RTE_MBUF_F_RX_L4_CKSUM_BAD
  • PKT_RX_IP_CKSUM_BAD $\rightarrow$ RTE_MBUF_F_RX_IP_CKSUM_BAD
  • PKT_TX_IP_CKSUM $\rightarrow$ RTE_MBUF_F_TX_IP_CKSUM
  • PKT_TX_IPV4 $\rightarrow$ RTE_MBUF_F_TX_IPV4
  • PKT_TX_TCP_CKSUM $\rightarrow$ RTE_MBUF_F_TX_TCP_CKSUM

Start Building

    Now we can start to build mTCP, just simply run make under the root path of mTCP repo, and we can get libmtcp.a under mtcp/lib if it is successfully compiled.
