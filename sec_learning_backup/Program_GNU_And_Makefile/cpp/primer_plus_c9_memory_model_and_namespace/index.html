1. 区分：作用域和链接性

    (1) 作用域 (scope)：描述了名称在单个文件内多大范围内可见。① 对于C++变量来说，作用域有局部和全局（文件作用域）两种。作用域为局部的变量只能在定义他的代码块里使用，为全局的变量在定义位置到文件结尾都能使用。② 对于C++函数来说，作用域可以是整个类和整个名称空间。
    (2) 链接性 (linkage)：描述了名称如何在不同文件间共享。
