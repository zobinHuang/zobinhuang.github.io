1. AJAX

    在本文中我们将描述如何在 JavaScript 中进行 HTTP 的请求和响应。如果您对 HTTP 部分的内容还不熟悉，我推荐你先看我的另一篇文章：万维网和 HTTP。

    AJAX 全名：Asynchronous JavaScript and XML。既不是一种语言，也不是一种框架，而是在阐述一种方法：在不重新加载整个页面的情况下，可以与服务器交换数据并更新部分网页内容。

    我们都知道，JavaScript 是一种在浏览器上运行的脚本语言。如果我们能够实现在 JS 中对服务器发起请求，并且将收到的响应更新到 DOM 中的话，我们就能够实现 AJAX。下面我们看看在 JS 中是如何实现的。

2.3 指定回调函数：readyState 和 status

    细心的读者会注意到我们在 open 函数中的第三个参数适用于指定我们的请求是一个异步的请求，这也是最通常的做法。那么，当我们调用完 send() 函数发送请求之后，当我们收到响应的时候，由于是异步的请求，那么我们该在哪里指定回调函数呢？本节我们将进行讨论。

    在 XMLHttpRequest 对象中，有两个非常重要的属性：readyState 和 status。

    readyState 用于存储 XMLHttpRequest 对象的状态，XMLHttpRequest 对象有以下五种状态：

状态编号 含义
0 未初始化
还没有调用send()方法
1 载入
已调用send()方法，正在发送请求
2 载入完成
send()方法执行完成，已经接收到全部响应内容
3 交互
正在解析响应内容
4 完成
响应内容解析完成，可以在客户端调用了

    status 属性对应于 HTTP 响应中的状态码。如：200 = "OK"，404 = "Not Found" 等。
