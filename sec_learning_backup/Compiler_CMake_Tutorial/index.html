target_xxx: 辅助生成目标

    cmake 的 目标 由 add_library 或者 add_executable 生成，前者仅完成编译，生成 library；后者完成编译+链接，生成可执行文件。

    下面我们对三个辅助生成目标的 cmake 脚本 API 进行总结:

  • target_include_directories cmake_doc_target_include_directories: 指定目标包含的头文件路径;
  • target_link_libraries cmake_doc_target_link_libraries: 指定目标链接的库;
  • target_compile_options cmake_doc_target_compile_options: 指定目标的编译选项;
