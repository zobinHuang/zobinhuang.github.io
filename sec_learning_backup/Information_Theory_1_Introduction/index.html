信息的基本概念

    客观世界三大要素：物质，能量，信息。

    1948 年，C.E.Shannon 发表了题为 “通信的数学理论” shannon 的论文，奠定了信息论的基础。

    信息从数量上等与通信前后 "不确定性" 的消除量 (i.e. 确定性的增加量)。

通信系统中的信息

    以下三个概念值得区分:

  • 信息 (Information) 指的是抽象的内容;
  • 消息 (Message) 指的是用来表达信息的某种客观对象 (e.g. 电话中的声音);
  • 信号 (Signal) 消息的表现形式，是消息的物理体现 (e.g. 光信号，电信号)

    信息论解答了通信理论中的两个问题:

  1. 临界数据压缩的值 (i.e. 熵);
  2. 临界通信传输速率的值 (i.e. 信道容量)

    通信系统的基本构成如下所示:

    信源 的功能是产生可能包含信息的消息。信源符号的概率空间可以表示为:

`(X,P) = ((x_1, x_2, x_3, ..., x_n),(p_1, p_2, p_3, ..., p_4))`

    采用合适的信源编码能通过 压缩码率 来提高传输有效性 (e.g. 经常出现的符号使用更短的代码来表示)，因此信源编码也称为 信源压缩编码。当然这种压缩是有限度的，我们在后面将会看到。

    编码器 可以细分为:

  • 信源编码器: 将信源消息变成符号，目的是提高传输有效性，也就是压缩每个信源符号传输所需要的代码的数目。可以把信源符号的概率空间表示如下:
  • 信道编码器: 给信源编码符号增加冗余符号，使得这些符号满足一定的数学规律，使得传输具有纠错或检错能力，目的是为了提高传输的可靠性。这些冗余符号不携带信息;
  • 调制器: 将信道编码符号调制为适合在信道上进行传输的信号。

    从有效性的角度来说，如果传输的各个信道编码更加独立，则它们各自携带的信息将更多; 从可靠性的角度来说，如果传输的编码更加相关 (i.e. 冗余)，那么系统将更能保证传输的编码内容的质量。

    信道: 分为无噪声/有噪声信道 (e.g. 加性, 乘性)，离散/离散时间连续/波形信道，无记忆/有记忆信道。

    在通信原理中，我们可以使用 `y(t) = \sum_{i=1}^{L}L_iS_I(t-t_i)=v(t)` 来描述一个多径信道。而在信息论中我们可以使用转移概率 `P(y_j|x_i)` 来描述信道，其含义是: 在已知发送方发出符号 `x_i` 的情况下，接收端辨识出 `y_j` 的概率。

香农信息理论的发展和应用

内容

    信源信息的度量: 信源限制为具有某一先验概率的随机过程，对信息进行度量是信息论的首要问题。

    无失真信源编码定理 (香农第一定理): 是信源压缩编码的理论基础。

    信道编码定理 (香农第二定理): 是信道压缩编码的理论基础。

    信息率失真定理 (香农第三定理): 是有损压缩编码的理论基础，其解决了有损压缩极限的理论问题，但未给出实际的有损信源编码的方法。

方法论

    以概率论、随机过程为基本研究工具，研究以编码器和译码器为重点的通信系统的整个过程，关心的是最优系统的性能和怎样到达这个新能。
