1. 概述

    当我们讨论 TCP 时，我们会把重点放在：TCP 是如何提供可靠的传输的

    香农在他 1948 年的工作里面帮助我们理解了：在有损耗的信道中移动的信息量的基本限制。有损信道意味着，数字信号在信道上将可能产生错误和丢失。信息编码 (coding theory) 领域提出来可以使用 纠错码 (error-correcting codes) 来纠正数字信息在信道上可能产生的错误比特。除了纠错码，另一种方案就是简单的让发端重传出错的信息，即 自动重传请求 (Automatic Repeat Request, ARQ)，这种方案是很多当下通信协议的基础，其中就包括了 TCP。
